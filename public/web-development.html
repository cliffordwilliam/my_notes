Web Development

Who?

They build and maintain websites. Work is project focused with a team. Clients:

  • Tech company
  • Organization
  • Governemnt

Types of Web Developers

  • Front end
    • presentation of content
    • user interface elements - navigation bar
    • HTML, CSS, JavaScript, and their relevant frameworks
  • Back end
    • stores and serves data from server for front end needs
    • Java, Python, Ruby and JavaScript
  • Full-stack
    • Can do Front end and Back end.

Careers

  • Large tech companies
    • Google, Meta and Amazon
    • strict requirements
    • excellent pay
  • Startups
    • very fast development pace
    • slightly lower salary
    • long hours
  • Freelancer
    • your own schedule and product
    • find your own clients
    • manage your own billing
    • need strong people skill
  • Web consultant
    • codes more than a freelancer
  • Large old companies
    • still need web developers
    • good work/life balance
    • slower than tech companies

Web Development Industry

The web development industry has a long history of successful developers with varying backgrounds, so people tend to care more about what you have actually built than how you got there. Contribute to open source projects like Odin. More contribution = good resume = more hireable
