Overall Structure

I first include css normalize the default styles for consistent styling baseline across browsers. Then I added an invisible border and set border box to everything. 

* {
    /* outline: 1px solid red; */
    border: 1px solid transparent;
    box-sizing: border-box;
}

Adding border to everything makes sure that all children margin do not collapse to parent's. Border box makes sure that the width and height that I set will not change when I add borders or padding. 

.content-anchor {
    padding-top: 1rem;
    padding-bottom: 1rem;
    border-top: 1px solid #ddd;
    color: #000;
    text-decoration: none;
    transition: ease-in-out, background-color .15s;
}

I always use padding top and bottom for things that will be in a flex column. 

.paper > * {
    padding-left: 1rem;
    padding-right: 1rem;
}

I add padding to the children and not the paper parent, this is so that the children are able to take full width and push its content with padding instead. This way I can have colored row in a paper. If paper had padding, then the elements are all pushed inside, it will not be able to have full width in the paper to be colored.

Production Testing & Well Clean Up

Test wells are drilled to check the potential of a reservoir. Flow capacity, this measures fluids (oil, gas, and water) flow rate from the reservoir under specific conditions. Reservoir and Flowing Pressures, reservoir pressure is underground, flowing is the pressure of fluid flowing from well. Flow potential, the max rate fluids can flow from well at specific conditions.

Overall, the purpose is to collect data about the reservoir's flow characteristics, productivity, and fluid composition. This data is needed for making decisions about production strategies, reservoir management, and optimizing the extraction of oil and gas resources.

Well Unloading & Blow Down

Process used to increase production for wells that are not optimal. This is because some wells may become idle wells / low pressure wells. These are not producing optimally. Tricky HTML

Margin Collapse

By default, children margin does not respect parent's. The best way to fix this issue is to add 1px padding to everything. However, siblings also do not respect each other's margin. The best way to fix this issue is to set the parent display to flex.

But I have come up with a strategy. Remember that padding respect each other. Margin doesn't. Use padding whenever it is okay to have the item take more space. Margin to push things away. There is no need for margin to be pushing each other, so always try to use one margin to push something away.

Border box

Default width and height do not include padding and border. When you set specific width and height, any padding or border added to it will make it fatter. If you set it to border box, the height and width you set will be respected always.
